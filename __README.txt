FILES ORGANIZER is a Python program designed to help people clean and organize their folders with just a few clicks. 



How to use it?

1. Copy the source path of the folder you wish to extract files and paste it into the first entry box SOURCE.

2. Copy the DESTINATION path of the folder where you wish to move all files and paste it into the second entry box DESTINATION.

3. Select the type of file you wish to extract from the source path to move to the destination folder path.

4. Click the button "move" to begin the action.
